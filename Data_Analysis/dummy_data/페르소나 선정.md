## 데이터 생성 방법

1. 각 페르소나별로 카테고리 선정 (메인 / 서브1 / 서브2로 분류)
2. 각 페르소나별로 20명 가량의 가상 User 생성
3. 각각의 User는 각각의 Category에 대해 랜덤한 범위의 값을 가지도록 설정 (8이내 / 5이내 / 3이내)
4. 아마도 총 유저는 900명 예상 / 카테고리는 423개 => (modeling을 위해 타협)
5. 테스트를 위해 outlier도 임의로 생성할 것
